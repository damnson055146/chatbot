# Ongoing Tasks

1. **会话与槽位前后端打通**：前端需改用 `/v1/slots` 与 `/v1/session` API 管理对话和槽位填充，并展示后端返回的 slot 提示/校验结果。
2. **文件上传与入库串联**：实现聊天面板的真实文件上传（调用 `/v1/upload`）、入库触发及状态反馈，并处理上传后的附件预览/引用。
3. **管理控制台**：在前端提供查看/调整检索参数、槽位配置、语料源、Prompt/模板、审计日志与 `metrics / status` 指标的界面。
4. **更人性化的提示词**：更新后端系统提示与槽位文案，让助手主动收集用户姓名、联系方式、目标与痛点，并在回答中引用已知信息。
5. **数据观测与告警**：补齐前端对 `/v1/status`、`/v1/metrics` 的可视化，方便排障与告警对接。
6. **测试与文档**：补充前后端单测/集测覆盖新增行为，同时更新 docs/README 说明新的流程与配置需求。
